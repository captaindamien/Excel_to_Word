--- Настройка конфигурационного файла ---

Конфигурационный файл нужен для указания программе откуда и куда передавать информацию с эксель таблицы в шаблон Word

Данные хранятся в строго типизированной форме типа:

[a] - имя переменной для шаблона в Word (обязательно буква или слово без цифр)

cell = A - ячейка в таблице Excel
Допускается ввод ячейки Excel вместе с номером, для точного сопоставления с шаблоном в формате D17

sheet = 1 - номер листа, где находится ячейка cell.

Так же программа может искать значения ячейки одного листа в другом, если у переменной присутствует параметр find_sheet.

find_sheet = 2 - номер листа, где нужно искать информацию.

Пример настройки конфигурационного файла:

[combobox]
combo_cell = C
combo_sheet = 1

[test]
cell = A
sheet = 1
find_sheet = 2

[combobox] - конфиг нужен для указания программе откуда и куда передавать информацию с эксель таблицы в поле "выбора нужной строки", а так же для упрощения пользователю в выборе по каким ячейкам работать циклу

combo_cell = A - ячейка в таблице Excel, в отличие от переменных, не допускается указывать ячейку вместе с номером

combo_sheet = 1 - номер листа, где находится ячейка cell.

--- Настройка шаблона Word ---

В самом шаблоне нужно только правильно расставить переменные, которые вы указали в конфигурационном файле [a] и т.д.:

Переменная имеет формат - {{имя_переменной}}, далее нужно ее выделить и пометить закладкой - Вставка -> Закладка, имя закладки должно быть строго такое же, как и у переменной.

Программа так же считывает Дату и Время и переводит их в нужный форма, если в Эксель таблице ячейка имеет формат даты. Ниже представлены варианты отображения даты и времени:

/* Пример: ahour, btime, cyear и т.д.*/

{{имя_переменнойhour}} - Выведет только час
{{имя_переменнойminute}} - Выведет только минуты
{{имя_переменнойtime}} - Выведет полное время вместе с секундами
{{имя_переменнойyear}} - Выведет только год
{{имя_переменнойmonth}} - Выведет только месяц
{{имя_переменнойday}} - Выведет только день
{{имя_переменнойdate}} - Выведет всю дату

Ниже представлен алгоритм работы поиска и выдачи результатов:
Берем значение из таблицы которое хранится в переменной ->
Ищем первое вхождение значения в другом листе (указан в имени переменной в конце) ->
Берем значение ячейки слева от найденного значения
(Если мы нашли значение в ячейке B5 листа 2, то на вывод идет значение ячейки А5 листа 2)
